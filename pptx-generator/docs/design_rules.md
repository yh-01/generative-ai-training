# 生成AI研修 PPTXデザインルール

## 目的
既存の生成AI研修スライドを、ページ数と意図を維持したまま、編集可能なPPTXとして再構成する。

## 対象者
中小企業の現場担当者。
定型的なPC作業はできるが、新しいPC作業やAI活用には不慣れ。
表現は平易にするが、チープにしない。

## デザイン基準
- メインカラー：#00AFB2
- 背景：白
- 薄背景：#EAFBFB または #E6F7F7
- 本文色：#1F2937 または #334155
- タイトル：黒、太字、大きめ
- 見出し：#00AFB2、太字
- 右上にセクションリボンを固定配置
- タイトル下に #00AFB2 の水平ラインを配置
- 外側カード：白背景、角丸大きめ、薄い影
- 内側カード：白背景、角丸、薄い枠線、薄い影
- 番号丸：#00AFB2 塗り、白文字
- アイコン：#00AFB2 の線画風
- 余白を広く取る
- 文章は「短い見出し＋1行説明」を基本とする

## 禁止事項
- ページ数を勝手に増やさない
- 元スライドの意図を変えない
- 派手な色を増やさない
- 長文をそのまま大きな箱に流し込まない
- チラシ風・初心者向けすぎる表現にしない
