# スライドパターン

## 左右比較
用途：
- 誤解 / 正しい考え方
- Before / After
- NG / OK

構成：
- タイトル
- リード文
- 左右2枚の大カード
- 各カードの中に3つの小カード
- 小カードは「番号丸 + アイコン + 見出し + 1行説明」

## 3カード
用途：
- メリット
- ポイント
- 注意点

構成：
- タイトル
- リード文
- 横並び3カード
- 各カードは「見出し + 2行以内の説明」

## プロンプト提示
用途：
- そのまま使えるプロンプト
- テンプレート
- 操作手順

構成：
- タイトル
- リード文
- 大きなコピーボックス
- 下部に3ステップの使い方

## 失敗例整理
用途：
- リスク
- 失敗パターン
- 注意喚起

構成：
- タイトル
- 左右または2段カード
- 「原因」「影響」「対策」などに分ける

## フロー / ステップ
用途：
- 手順
- プロセス
- 学習の流れ

構成：
- タイトル
- リード文
- 横並びステップ
- 矢印または番号で流れを示す
